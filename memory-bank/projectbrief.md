# Project Brief: Fzt. Ezgi Acem Website

## Project Overview
Professional website for **Fzt. Ezgi Acem** - a Pediatric Physiotherapist based in Adana, Turkey, specializing in sensory integration therapy for children.

## Core Requirements

### Primary Goals
1. **Professional Online Presence** - Showcase expertise in pediatric physiotherapy
2. **Patient Acquisition** - Generate leads through contact forms and WhatsApp integration
3. **Educational Content** - Blog posts about child development, sensory integration, and therapy
4. **Admin Management** - Self-service content management via custom admin panel

### Target Audience
- Parents of children with developmental concerns
- Parents seeking sensory integration therapy
- Healthcare professionals for referrals
- Location: Adana, Turkey and surrounding areas

### Key Features
- Responsive, mobile-first design with child-friendly aesthetics
- Blog system with rich text editor and SEO fields
- Admin panel for content management (posts, specializations, settings)
- Contact form with WhatsApp integration
- SEO optimized for local search (Adana fizyoterapist)

## Technical Requirements
- **Performance**: Fast loading, optimized images (WebP)
- **SEO**: Schema markup (MedicalBusiness), meta tags, sitemap
- **Security**: HTTPS, secure admin authentication
- **Reliability**: Auto-restart services, proper error handling

## Success Metrics
- Google PageSpeed score > 90
- Local SEO ranking for "Adana pediatrik fizyoterapist"
- Admin can easily manage all content
- New blog posts immediately visible (SSR)

## Constraints
- Budget-conscious hosting (single VPS)
- Owner manages content independently
- Turkish language primary
